Sobre el proyecto

DataConecta combina Machine Learning con visualizaciones impactantes para detectar fraudes en transacciones con tarjetas de crédito. Utiliza Random Forest, SMOTE y análisis detallado para ofrecer métricas como AUC, precisión y recall.

  • Scikit-Learn (Random Forest, SMOTE)
  • Matplotlib • Seaborn para gráficas
  • Bootstrap 5 • GitHub Pages para la web
