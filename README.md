# Dataconecta — Plataforma Integral de Consultoría y Colaboración

Sitio web avanzado con funcionalidades completas de automatización, IA, colaboración, seguridad y sostenibilidad para consultoría en ciencia de datos.

## 🚀 Funcionalidades Avanzadas Implementadas

### ✅ **Automatización de Procesos**
- Workflows automatizados para procesamiento de datos
- Auto-guardado inteligente de formularios
- Monitoreo de rendimiento en tiempo real
- Pruebas automatizadas continuas

### ✅ **Inteligencia Artificial**
- Asistente virtual con procesamiento de lenguaje natural
- Base de conocimiento interactiva
- Respuestas contextuales sobre servicios y procesos
- Historial de conversaciones persistente

### ✅ **Visualización Avanzada de Datos**
- Dashboard interactivo con métricas en tiempo real
- Gráficos de rendimiento, analítica de usuarios y conversiones
- Proyecciones de ingresos y embudo de ventas
- Exportación de datos y pantalla completa

### ✅ **Seguridad Avanzada**
- Content Security Policy (CSP) automático
- Protección XSS multicapa
- Gestión de sesiones con timeout automático
- Cifrado de localStorage y cumplimiento GDPR
- Banner de privacidad con consentimiento granular

### ✅ **Framework de Testing Integral**
- Pruebas unitarias y de integración automatizadas
- Testing de accesibilidad WCAG 2.1
- Pruebas de rendimiento y Core Web Vitals
- Regresión visual y compatibilidad cross-browser
- Tests manuales personalizables

### ✅ **Plataforma de Colaboración**
- Workspaces compartidos en tiempo real
- Base de conocimiento comunitaria
- Foro de discusión categorizado
- Sistema de archivos compartidos
- Notificaciones de actividad

### ✅ **Gestión de Sostenibilidad**
- Tracking de huella de carbono en tiempo real
- Métricas de eficiencia energética
- Objetivos de sostenibilidad personalizables
- Modo eco para reducir consumo
- Compensación de carbono virtual

## 🎯 Características Principales

- **Multilenguaje:** Español/Inglés (botón arriba a la derecha)
- **Modo claro/oscuro** con detección automática
- **Navegación intuitiva** entre páginas
- **Responsive design** optimizado para todos los dispositivos
- **Progressive Web App** con funcionalidades offline

## 🛠️ Tecnologías Utilizadas

- **Frontend:** HTML5, CSS3, JavaScript ES6+
- **Framework CSS:** Bootstrap 5.3.2
- **Iconos:** Bootstrap Icons
- **Fuentes:** Google Fonts (Montserrat, Inter)
- **APIs:** Canvas API, Intersection Observer, Performance Observer
- **Almacenamiento:** localStorage, sessionStorage
- **Seguridad:** CSP, XSS Protection, Encryption

## 🚀 Inicio Rápido

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/dsd228/Dataconecta.git
   cd Dataconecta
   ```

2. **Inicia un servidor local:**
   ```bash
   python3 -m http.server 8000
   # o
   npx serve .
   ```

3. **Abre en el navegador:**
   ```
   http://localhost:8000
   ```

## 🎮 Cómo Usar las Nuevas Funcionalidades

### Asistente de IA
- Haz clic en el botón flotante azul (🤖) en la esquina inferior derecha
- Usa las sugerencias rápidas o escribe tus propias preguntas
- El asistente responde sobre servicios, procesos y contacto

### Dashboard de Analytics
- Haz clic en "Analytics" en la navegación superior
- Visualiza métricas de rendimiento, usuarios y conversiones
- Usa los controles para actualizar, exportar o ver en pantalla completa
- Presiona Escape para cerrar

### Panel de Testing
- Haz clic en el botón flotante rojo (🐛) para abrir las pruebas
- Ejecuta pruebas automatizadas, de rendimiento o accesibilidad
- Crea y ejecuta tests manuales personalizados
- Exporta reportes de calidad

### Colaboración
- Haz clic en el botón flotante verde (👥) para colaborar
- Accede a workspaces, base de conocimiento y foro
- Comparte archivos y participa en discusiones
- Recibe notificaciones de actividad

### Sostenibilidad
- Haz clic en el botón flotante verde (🌱) para ver métricas ambientales
- Monitorea tu huella de carbono en tiempo real
- Activa el modo eco para reducir consumo energético
- Establece objetivos de sostenibilidad

## 📊 Métricas y Monitoreo

El sistema incluye monitoreo automático de:
- **Performance:** Tiempo de carga, FCP, LCP, CLS
- **Seguridad:** Eventos de seguridad y amenazas
- **Sostenibilidad:** Emisiones CO₂ y eficiencia energética
- **Calidad:** Cobertura de pruebas y accesibilidad
- **Uso:** Interacciones de usuario y analítica

## 🔒 Privacidad y Seguridad

- **Cumplimiento GDPR** con banner de consentimiento
- **Encriptación** de datos locales sensibles
- **Protección XSS** con sanitización automática
- **CSP** para prevenir inyección de código
- **Auditoría** completa de eventos de seguridad

## 🌱 Sostenibilidad

Este sitio web está diseñado con principios de sostenibilidad:
- **Optimización de recursos** para reducir transferencia de datos
- **Lazy loading** para mejorar eficiencia energética
- **Modo eco** que reduce animaciones y efectos
- **Tracking de CO₂** para consciencia ambiental
- **Compensación virtual** mediante plantación de árboles
